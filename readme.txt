== Frequently Asked Questions ==

= Is liveweb free? =

The liveweb offers you a free trial of 14 days. It is currently in early access, and we are welcoming new users by request.

= What does liveweb let me do? =

liveweb adds a widget to your site which visitors can use to engage with you. You can use it to collect feedback, learn about your visitors, and even close deals. Instantly connect with your customers - with secure HD audio/video calls with screen and document sharing hosted directly at your website.

= Do I have to install any software on my server to get this working? =

Not at all! liveweb is a hosted live chat and call service. You need to configure the plugin, and you are ready!

= Which web browsers work best with this plugin =

Though designed to work on most browsers, liveweb works best in the following environment: Firefox 2 or later (Mac, PC, or Linux), Safari 2 or later (Mac), Google Chrome (PC, Mac).


== Usage ==

After enabling the plugin, head to the widget customization page to change settings and integrate its look and feel to match your site. When done, get it running by copying your API key to the liveweb WordPress plugin.

== Installation ==

*Server Requirements:* PHP4 or PHP5.

*WordPress versions:* WordPress 4.9 and up.

Note: You will need to request early access for a liveweb account : <a href="https://liveweb.io">Request access here!</a>.

Step-by-step Guide:

* Install plugin from WordPress directory and activate it.
* Under left sidebar, click on liveweb to add your liveweb API key.
* Customize the chat widget to your preference.
* Finally, make full use of our Dashboard to manage your chat widget.
